Estrutura de um Documento HTML

A estrutura de um documento HTML é composta por três elementos principais: html, head e body.

Exemplo de Código com uma Estrutura Simples de um Arquivo HTML:

<html>
<head>
</head>
<body>
</body>
</html>

A tag contém todas as informações do documento HTML.

É importante ressaltar que as tags sempre começam com < > e terminam com . O / é utilizado para indicar o final de uma tag, ajudando a manter a organização do código.

A tag contém todas as informações sobre o site, como o título do documento HTML e outros metadados. Abaixo, você verá um exemplo de uma estrutura mais robusta de um documento HTML.

A tag contém todo o conteúdo da página do seu site. É a parte que o usuário verá ao acessar a página.

É comum, ao criar um projeto HTML, nomear o arquivo principal como index.html. Essa prática é amplamente utilizada entre desenvolvedores para indicar o início ou a página principal de um site. No entanto, isso não é obrigatório e depende da sua escolha ou das necessidades do seu projeto.

Exemplo de Código com uma Estrutura mais Robusta de um Arquivo HTML:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
</body>
</html>

Novas Tags e Atributos no HTML

<!DOCTYPE html>

A declaração é a primeira linha de código em um documento HTML e serve para informar ao navegador que o documento está escrito em HTML5. Essa declaração é essencial, pois garante que o navegador interprete o código corretamente de acordo com as normas do HTML5.

<meta charset="UTF-8">

A tag especifica a codificação de caracteres usada no documento HTML. O valor UTF-8 é o mais comum e suporta uma ampla gama de caracteres, incluindo letras, números e símbolos de diferentes idiomas. Isso garante que o conteúdo do site seja exibido corretamente, independentemente do idioma ou dos caracteres utilizados

<meta name="viewport" content="width=device-width, initial-scale=1.0">

A tag é crucial para garantir que o layout do seu site seja responsivo, ou seja, que ele se adapte corretamente a diferentes tamanhos de tela, como em dispositivos móveis. O atributo content="width=device-width, initial-scale=1.0" indica ao navegador que a largura da viewport deve ser igual à largura do dispositivo e que o nível de zoom inicial é 1.0, ou seja, sem zoom.

<title>Document</title>

A tag <title> define o título do documento HTML, que é exibido na aba do navegador ou na barra de título da janela. Este título é importante tanto para a experiência do usuário quanto para a otimização de mecanismos de busca (SEO), pois ele aparece nos resultados de pesquisa.

O que são Tags em HTML

Tags em HTML:

Tags são comandos em HTML que servem para adicionar e estruturar elementos em um documento. Existem tags para texto, imagens, links e muitos outros elementos que ajudam a compor o conteúdo de uma página web.

Tipos de Tags

Texto

As tags de texto em HTML são variadas, permitindo a formatação e a organização do conteúdo textual de diferentes maneiras. A seguir, alguns exemplos importantes:

  • Tag <h1>; até <h6>;
    • Utilizadas para criar títulos e subtítulos.
    • <h1> é o título principal e de maior destaque, enquanto <h6> é o de menor destaque.
    • Quanto maior o número, menor é o tamanho e a importância do título.
    • Exemplo:

      • <h1>Este é um Título Principal</h1>
      <h2>Este é um Subtítulo</h2>

  • Tag <p>:
    • Utilizada para adicionar parágrafos ao documento.
    • Exemplo:

      • <p>Este é um parágrafo de texto.</p>
