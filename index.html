Estrutura de um Documento HTML

A estrutura de um documento HTML é composta por três elementos principais: html, head e body.

Exemplo de Código com uma Estrutura Simples de um Arquivo HTML:

<html>
<head>
</head>
<body>
</body>
</html>

A tag contém todas as informações do documento HTML.

É importante ressaltar que as tags sempre começam com < > e terminam com . O / é utilizado para indicar o final de uma tag, ajudando a manter a organização do código.

A tag contém todas as informações sobre o site, como o título do documento HTML e outros metadados. Abaixo, você verá um exemplo de uma estrutura mais robusta de um documento HTML.

A tag contém todo o conteúdo da página do seu site. É a parte que o usuário verá ao acessar a página.

É comum, ao criar um projeto HTML, nomear o arquivo principal como index.html. Essa prática é amplamente utilizada entre desenvolvedores para indicar o início ou a página principal de um site. No entanto, isso não é obrigatório e depende da sua escolha ou das necessidades do seu projeto.

Exemplo de Código com uma Estrutura mais Robusta de um Arquivo HTML:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
</body>
</html>

Novas Tags e Atributos no HTML

<!DOCTYPE html>

A declaração é a primeira linha de código em um documento HTML e serve para informar ao navegador que o documento está escrito em HTML5. Essa declaração é essencial, pois garante que o navegador interprete o código corretamente de acordo com as normas do HTML5.

<meta charset="UTF-8">

A tag especifica a codificação de caracteres usada no documento HTML. O valor UTF-8 é o mais comum e suporta uma ampla gama de caracteres, incluindo letras, números e símbolos de diferentes idiomas. Isso garante que o conteúdo do site seja exibido corretamente, independentemente do idioma ou dos caracteres utilizados

<meta name="viewport" content="width=device-width, initial-scale=1.0">

A tag é crucial para garantir que o layout do seu site seja responsivo, ou seja, que ele se adapte corretamente a diferentes tamanhos de tela, como em dispositivos móveis. O atributo content="width=device-width, initial-scale=1.0" indica ao navegador que a largura da viewport deve ser igual à largura do dispositivo e que o nível de zoom inicial é 1.0, ou seja, sem zoom.

<title>Document</title>

A tag <title> define o título do documento HTML, que é exibido na aba do navegador ou na barra de título da janela. Este título é importante tanto para a experiência do usuário quanto para a otimização de mecanismos de busca (SEO), pois ele aparece nos resultados de pesquisa.

O que são Tags em HTML

Tags em HTML:

Tags são comandos em HTML que servem para adicionar e estruturar elementos em um documento. Existem tags para texto, imagens, links e muitos outros elementos que ajudam a compor o conteúdo de uma página web.

Tipos de Tags

Texto

As tags de texto em HTML são variadas, permitindo a formatação e a organização do conteúdo textual de diferentes maneiras. A seguir, alguns exemplos importantes:

  • Tag <h1>; até <h6>;
    • Utilizadas para criar títulos e subtítulos.
    • <h1> é o título principal e de maior destaque, enquanto <h6> é o de menor destaque.
    • Quanto maior o número, menor é o tamanho e a importância do título.
    • Exemplo:

      • Este é um Título Principal

      Este é um Subtítulo

  • Tag <p>:
    • Utilizada para adicionar parágrafos ao documento.
    • Exemplo:

      • Este é um parágrafo de texto.

  • Tag <blockquote>
    • Utilizada para adicionar uma citação em bloco. Normalmente, o texto dentro de <blockquote> é recuado em relação ao resto do conteúdo.
    • Exemplo:

      • Esta é uma citação de um autor famoso.

  • Tag <i>
    • Utilizada para adicionar texto em itálico, geralmente para enfatizar uma palavra ou frase.
    • Exemplo:

      • Este texto está em itálico.

  • Tag <u>
    • Utilizada para sublinhar texto.
    • Exemplo:

      • Este texto está sublinhado.

  • Tag <strong>
    • Utilizada para destacar um texto em negrito, indicando uma importância maior.
    • Exemplo:

      • Este texto está em negrito.

  • Tag <mark>
    • Utilizada para destacar texto com um fundo amarelo, como se estivesse marcado com um marcador de texto.
    • Exemplo:

      • Este texto está destacado.

  • Tag <sup>
    • Utilizada para criar texto sobrescrito, comum em fórmulas matemáticas ou notações científicas.
    • Exemplo:

      • H2O

  • Tag <input>:
    • A tag <input> é usada para criar diferentes tipos de campos de entrada em formulários, como caixas de texto, botões de rádio, checkboxes, entre outros.
    • Como não contém conteúdo entre a abertura e o fechamento, a tag <input> é auto-fechadas.

O que são Atributos em HTML

Atributos:

Atributos em HTML são usados para fornecer informações adicionais sobre um elemento. Eles são inseridos dentro da tag de abertura de um elemento e são compostos por um nome de atributo e um valor atribuído a ele. Atributos podem controlar a aparência, o comportamento e o conteúdo de um elemento HTML.

Tipos de Atributos

Atributos Globais:

Atributos globais são atributos que podem ser aplicados a qualquer tag HTML. Eles oferecem funcionalidades comuns que podem ser usadas em diversos elementos do HTML.

  • Atributo id:
    • O atributo id é utilizado para atribuir um identificador único a um elemento HTML. Esse identificador deve ser único em todo o documento, e é muito útil para aplicar estilos específicos via CSS ou para manipular o elemento através de JavaScript.
    • Exemplo:

      • <div id="cabecalho">Cabeçalho do site</div>

  • Atributo class:
    • O atributo class é utilizado para agrupar elementos em classes, permitindo aplicar estilos ou comportamentos comuns a vários elementos. Diferente do id, uma classe pode ser reutilizada em múltiplos elementos.
    • Exemplo:/li>

      <p class="texto-destaque"> Este é um parágrafo com destaque.

  • Atributo style:
    • O atributo style permite aplicar estilos CSS diretamente a um elemento HTML. No entanto, o uso de style inline não é recomendado para grandes projetos, pois dificulta a manutenção e separação de responsabilidades.
    • Exemplo:

      • <h1 style="color: blue; text-align: center;">Título Centralizado em Azul</h1>

Atributos Específicos:

Atributos específicos são aqueles que se aplicam apenas a determinadas tags e controlam características específicas desses elementos.

  • Atributo src:
    • O atributo src (source) é usado na tag <img> para especificar o caminho da imagem a ser exibida, ou na tag <script> para especificar o caminho de um arquivo JavaScript.
    • Exemplo:

      • <img src="imagem.jpg" alt="Descrição da imagem">

  • Atributo width:
    • O atributo width define a largura de um elemento, como uma imagem ou um vídeo. O valor pode ser especificado em pixels (px) ou em porcentagem (%) em relação ao elemento pai.
    • Exemplo:

      • <img src="imagem.jpg" alt="Descrição da imagem" width="300px">

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de um documento escrito em HTML ou XML. CSS define como os elementos HTML devem ser exibidos na tela, em papel ou em outros meios. Ele permite a separação entre o conteúdo do documento (HTML) e sua apresentação visual, tornando o código mais limpo e fácil de manter.
